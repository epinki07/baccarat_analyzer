# AGENTS.md

## Objetivo
Construir una app local de análisis de baccarat en Python + Streamlit.
Debe enfocarse en estadísticas, visualización, bankroll y alertas.
No debe prometer predicción segura ni ganancias.

## Flujo de trabajo
1. Planear primero
2. Implementar por fases
3. Ejecutar pruebas después de cada fase
4. Corregir errores antes de continuar
5. Entregar proyecto ejecutable

## Stack
- Python 3.11+
- Streamlit
- pandas
- plotly o matplotlib
- pytest

## Estructura esperada
- app.py
- requirements.txt
- README.md
- src/
- tests/
- sample_data/

## Comandos
- pip install -r requirements.txt
- pytest
- streamlit run app.py

## Reglas
- No afirmar que existe predicción segura
- No usar lenguaje de “ganancia garantizada”
- Incluir advertencia sobre ventaja de la casa
- Mantener código modular y legible
- Añadir manejo de errores
- No agregar dependencias innecesarias

## Definición de terminado
- Carga CSV
- Muestra métricas
- Genera gráficos
- Simula bankroll
- Genera alertas
- Exporta resultados
- Tests básicos pasan
- README completo
